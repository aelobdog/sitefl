SiTeFL

The Simple Text Formatting Language

SiTeFL is a text formatting language like Markdown, with a text-to-html tool written from scratch in Go.

Syntax:

  1| Headings    :   # (1 -> 6) 
  2| 
  3| Bold        :   *bold*
  4| 
  5| Italics     :   /italics/
  6| 
  7| Underline   :   _underline_
  8| 
  9| Code        :   `this is code`
 10| 
 11| Images      :   ![Alt-text::width::height](link to image)
 12| 
 13| Link        :   @[Display-text](link url)
 14| 
 15| NewLine     :   ;;
 16| 
 17| Horizontal  :   ---
 18| 
 19| Escape
 20| Formatting  :   { *not bold* will not appear as bold ! }
 21| (BLOCK)
 22| 
 23| Escape
 24| Single      :   \\
 25| Character


Usage:

  1| sitefl [-OPTIONS] [stylesheet, template] source destination


Options:

  1| f : Prints this message
  2| n : Preserves new lines
  3| s : attach Stylesheet
  4| t : use template
  5| w : wrap the output within a div (id = 'unit')

Source:

  1| filename : file to obtain input from
  2| 'in' : get input from stdin
  3| 
  4| (enables : grep -o "something" | sitefl in DESTINATION )

Destination:

  1| filename : file to send output to
  2| 'out' : send output to stdout
  3| 
  4| (enables : sitefl SOURCE out | grep "something" )

Example:

  1| sitefl -nts defaults/templateHTML.html defaults/templateCSS.css index.stfl index.html
